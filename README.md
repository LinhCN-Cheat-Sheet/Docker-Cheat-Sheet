## Container
### Lifecycle
Creates a container but does not start it
    > docker create 
Allows the container to be renamed
    > docker rename 
Creates and starts a container in one operation
    > docker run 
Deletes a container
    > docker rm 
updates a container's resource limits
    > sdocker update
